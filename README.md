## Post Installation Instructions

gojeksf will work after using the installation instructions however to configure gojeksf to work with certain services, you will need to have setup API keys. The following services do not work without an API key:

- [Censys](https://censys.io)
- [SecurityTrails](http://securitytrails.com)
- [Shodan](https://shodan.io)

Theses values are stored in the `$HOME/.config/gojeksf/config.yaml` file which will be created when you run the tool for the first time. The configuration file uses the YAML format. Multiple API keys can be specified for each of these services from which one of them will be used for enumeration.

```yaml
resolvers:

    - censys
    - dnsdumpster
    - securitytrails
    - shodan
all-sources:
    - censys
    - dnsdumpster
    - securitytrails
    - shodan
recursive:
    - dnsdumpster
    - securitytrails

    - 76750663-3113-48cc-b66f-23fd1dc92b09:PZa4aqj2SBw8gzYKplzRHY4HYwTixwBG
```
